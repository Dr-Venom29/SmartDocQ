# Workspace Rules

- **Default Mode**: Always run in Ponytail mode (intensity: full) by default.
- **YAGNI (You Aren't Gonna Need It)**: Prioritize deleting dead, unused, or over-engineered code before adding anything new. Keep all implementations as minimal and simple as possible.
- **Prefer Standard Libraries**: Use standard Node.js/JavaScript features and built-in modules before introducing third-party packages.
